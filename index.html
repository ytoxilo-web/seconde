Questions frequentes

Une section pratique pour expliquer le fonctionnement quand la plateforme aura plus de contenu.

Quand les cours seront-ils ajoutes ?

Les cartes sont deja prevues pour accueillir les premiers cours. Il suffira de remplacer les exemples par les vrais contenus.

Peut-on ajouter d'autres matieres ?

La liste actuelle est fixee sur les matieres principales du site. Les cours peuvent etre ajoutes dans chacune depuis le panel admin.

Le site fonctionne-t-il sur mobile ?

Oui. Les sections passent automatiquement en une colonne sur les petits ecrans pour garder une lecture confortable.
